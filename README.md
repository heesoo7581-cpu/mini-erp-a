# Mini ERP A Starter

가공업체용 미니 ERP A안의 초기 프로젝트 구조입니다.

## 포함 내용
- 대시보드
- 거래처 관리
- 품목 관리
- 수주 등록
- 작업지시 등록
- 생산실적 등록
- 출하 등록
- 거래명세서 출력 와이어프레임
- 권한/코드 관리 화면

## 실행 방법
```bash
npm install
npm run dev
```
